Удаление пользователя из базы доступа

Удаление проходит если совпадают поля username password realm

  • Curl
  • HTTP
curl -X POST -H "Content-Type: application/json" -d '{
		"secretKey" : "talkoutSuperSecretKey2",
		"command": "REMOVE_USER",
		"users":[
			
				{
                "username": "test",
                "password": "foobar",
                "realm": "example.org",
                "mode": "REFUSED"
            }
			]
}' "http://turn1.talkout.me:8765/"
POST / HTTP/1.1
Host: turn1.talkout.me:8765
Content-Type: application/json

{
		"secretKey" : "talkoutSuperSecretKey2",
		"command": "REMOVE_USER",
		"users":[
			
				{
                "username": "test",
                "password": "foobar",
                "realm": "example.org",
                "mode": "REFUSED"
            }
			]
}

Изменение поля пользователя

Изменение проходит если совпадают поля username password realm

изменяется поле mode

Типы mode

authorized (user authorized)

refused (user denied access)

restricted (for when bandwidth limiting is implemented)
